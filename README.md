# Digital-Clock
A sleek, minimalist digital clock built with vanilla JavaScript and CSS Grid.

- Real-Time Updates: Uses JavaScript setInterval to refresh every second without page reloads.
- Minimalist UI: Features a dark, distraction-free theme and elite typography that aligns with a professional aesthetic.
- Responsive Design: Fully centered and compatible across all screen sizes using CSS Flexbox.

🛠️ Technologies

- HTML5
- CSS3 
- JavaScript
